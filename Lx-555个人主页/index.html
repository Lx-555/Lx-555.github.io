项目展示

AI知识库问答系统

技术栈：Python + FastAPI + LangChain + ChatGLM-6B + MySQL

一款面向企业的私有化知识库问答工具，支持上传文档（PDF/Word/TXT），通过LangChain实现文档拆分、向量存储，结合ChatGLM-6B大模型实现精准的上下文问答。

  • 支持多种文档格式解析，自动提取文本并生成向量嵌入
  • 私有化部署，数据本地存储，保障企业信息安全
  • 自定义问答模板，适配不同业务场景（如客服、技术支持）
GitHub仓库 在线演示（暂未部署）

AI代码助手工具

技术栈：Vue3 + Node.js + OpenAI API + TypeScript

  • 支持Python/Java/JavaScript等10+编程语言
  • 本地缓存对话记录，支持自定义API密钥
  • 一键复制代码、格式化输出，提升开发效率
